# Tic Tac Toe Web Game

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features
- Two player game (X and O)
- Winner detection
- Reset game option
- Responsive layout

## Technologies Used
- HTML
- CSS
- JavaScript
